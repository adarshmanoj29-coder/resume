# Resume Website (Single Page)

This is a simple, fast resume website (HTML + CSS only).

## Files
- `index.html`
- `styles.css`
- (optional) `Adarsh_Manoj_CV.pdf` — put your CV PDF here to enable the download button.

## Quick local preview
Just open `index.html` in your browser.

## Publish (GitHub Pages)
1. Create a new repo on GitHub (e.g., `resume`).
2. Upload these files to the repo root.
3. Go to **Settings → Pages**
4. **Source:** Deploy from a branch
5. **Branch:** `main` / `(root)`
6. Save — your site will be available at:
   `https://<your-username>.github.io/resume/`

## Publish (Netlify)
1. Drag-and-drop the folder into Netlify “Add new site”.
2. It will give you a public URL immediately.
